About this web page

BookSphere lets users search for books, browse results in multiple views, inspect full book details from the Google Books API, and save a personal reading list. It is made using HTML, CSS, JavaScript, and jQuery.

Project disclaimer: This website is a student project made for educational reasons as part of a university course.
